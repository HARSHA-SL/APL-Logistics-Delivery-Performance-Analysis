# APL Logistics – Delivery Performance Analysis

## Project Objective
To evaluate SLA compliance, diagnose delay drivers, and assess financial impact across shipping modes and regions.

## Key Findings
- 57% shipments classified as delayed
- Majority delays limited to 1 day
- Premium shipping modes exhibit SLA misalignment
- Standard Class contributes highest delay volume
- Moderate financial impact per order

## Tools Used
- Python (Pandas, NumPy)
- Streamlit
- Exploratory Data Analysis

- Exploratory Analysis Notebook
- Final Structured Analysis Notebook
